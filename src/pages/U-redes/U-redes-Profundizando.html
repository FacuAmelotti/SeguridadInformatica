¿Que significa DNS?

  • ● DNS (Domain Name System) → Es un sistema que traduce nombres de dominio (como google.com) en direcciones IP.
  • ● Funciona como la "agenda de contactos" de Internet.
  • ● Ejemplo de dirección IP: En lugar de recordar 142.250.190.14, simplemente escribes google.com.
  • El DNS encuentra la IP correcta y te conecta con el servidor.
nslookup google.com #Comando para ver la dirección IP asociada al dominio consultado.
tracert google.com #Comando para ver la ruta que siguen los paquetes hasta llegar al servidor.

El DNS es como una guía telefónica que traduce nombres en direcciones IP.....
